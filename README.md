Hospital Emergency Room Dashboard (Power BI)
-> Project Overview:

The Hospital Emergency Room Dashboard is built using Power BI to analyze patient visits, waiting time, and satisfaction scores in a hospital emergency department.

The goal of this project is to help hospital administrators and healthcare professionals monitor operational performance and identify patterns in patient visits to improve service efficiency.

-> Objectives:

Analyze patient visit trends

Monitor average waiting time

Evaluate patient satisfaction scores

Identify peak hours and busiest days

Understand demographic patterns of ER visits

Tools & Technologies:

Power BI – Data visualization and dashboard creation

Excel / CSV Dataset – Data source

Data Cleaning & Transformation – Power Query

DAX – Calculated measures and KPIs

-> Dataset Information:

The dataset contains hospital emergency room visit records including:

Patient visit date and time

Patient gender

Age group

Race

Department referral

Waiting time

Satisfaction score

📈 Dashboard Features
1️. Key Performance Indicators (KPIs)

Total Patient Visits

Average Waiting Time

Average Satisfaction Score

2️. Patient Visit Analysis

Visits by gender

Visits by age group

3️. Patient Satisfaction Analysis

Satisfaction score by race

Rating visualization

4️. Time-Based Analysis

Patient visits by day and hour

Identification of busiest day and time

5️. Department Referral Analysis

Average waiting time by hospital department

-> Key Insights:

Tuesday shows the highest patient traffic

Peak visit time occurs around 4 PM

Certain departments experience longer waiting times

Patient satisfaction varies across demographic groups
